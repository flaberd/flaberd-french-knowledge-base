# Agent Instructions

## Purpose

This repository is the source of truth for French learning progress.

It contains structured knowledge extracted from lessons, PDFs, notes, exercises, and other learning materials.

The repository should always reflect:

* what has been studied;
* what is currently being studied;
* how knowledge has evolved over time;
* which grammar, vocabulary, verbs, topics are available for practice.

## Core Principle

Lesson files are historical records.

Canonical knowledge files are the source of truth.

Information extracted from a lesson must always be merged into the appropriate knowledge files instead of remaining only inside the lesson record.

## Processing Rules

Follow instructions/workflow.md 

## File Creation

Before creating a new file:

1. Search for an existing file covering the same topic.
2. Update the existing file whenever appropriate.
3. Create a new file only if the topic does not already exist.
