# Basic Information

## French

ce (cette, cet, ces)

## Part of Speech

Determiner

## Ukrainian

цей / ця / це / ці (вказівний прикметник)

## English

this / that (demonstrative adjective)

# Forms *(if applicable)*

## Masculine

ce (cet before a vowel or mute h)

## Feminine

cette

## Plural

ces

---

# Pronunciation *(if applicable)*

- **ces** + vowel-initial noun → liaison in [z]: *ces_arbres*, *ces_hommes*.
- "Ce" becomes "cet" before a masculine noun starting with a vowel or mute "h": *ce acteur* → **cet acteur** ; *ce homme* → **cet homme**.

---

# Examples

- Regardez : ce chat / cette voiture / ces arbres.
- Regarde cette fourmi (sur le sol). Regarde cette étoile (dans le ciel).
- Ce livre est intéressant. Cette histoire est intéressante. Ces livres sont intéressants. Ces histoires sont intéressantes. *(model given in source)*

---

# Notes

The demonstrative agrees with the noun it accompanies, like the possessive adjective — see [vocabulary/determiners/mon.md](mon.md).
