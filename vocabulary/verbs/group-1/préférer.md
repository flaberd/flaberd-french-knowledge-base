# Forms *(if applicable)*

Group 1 (regular -er)

# Conjugation *(if applicable)*

Present tense: je préfère, tu préfères, il/elle/on préfère, nous préférons, vous préférez, ils/elles préfèrent.

---

# Irregularities *(if applicable)*

Stem-changing verb: é→è in all forms except nous/vous. See [grammar/er-verbs-stem-changes.md](../../../grammar/er-verbs-stem-changes.md).

# Examples

- Je préfère le miel. Tu préfères quoi? Il/Elle/On préfère la confiture. Nous préférons le chocolat noir. Vous préférez rester à la maison? Elles/Ils préfèrent les vacances à la mer.
- Vous préférez les pâtes ou le riz? - Je préfère le riz.
