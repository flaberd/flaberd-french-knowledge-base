# Forms *(if applicable)*

# Conjugation *(if applicable)*

Present tense: je choisis, tu choisis, il/elle/on choisit, nous choisissons, vous choisissez, ils/elles choisissent. See [grammar/second-group-verbs-present.md](../../../grammar/second-group-verbs-present.md).

# Notes

Listed in the source both as a classification example of the 2e groupe and in the "which group?" exercise; its present-tense conjugation is now taught (see [lesson_00019](../../../lessons/lesson_00019.md)).

The partitif exam in [lesson_00042](../../../lessons/lesson_00042.md) flags *choisir* with an emphatic "2e groupe!!!" note on both items that use it, and the student got both wrong — a recorded common mistake, likely conjugating it as if it were a 1er groupe (-er) verb instead of using the -issons/-issez/-issent pattern.
