# Examples

- Le petit chat (plaire) ___ à mes parents. *(passé composé drill: "a plu")*

# Notes

Present-tense conjugation not yet taught; auxiliary is **avoir**. Used impersonally/with an indirect object like *aimer* in reverse ("X plaît à Y" = Y likes X).
