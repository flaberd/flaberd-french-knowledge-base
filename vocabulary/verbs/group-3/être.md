# Basic Information

## French

être

## Part of Speech

Verb

## Ukrainian

бути

## English

to be

# Forms *(if applicable)*

## Verb Group

Group 3 (irregular)

---

# Conjugation *(if applicable)*

Present tense:

| Pronoun | Form |
|---|---|
| Je | suis |
| Tu | es |
| Il / Elle / On | est |
| Nous | sommes |
| Vous | êtes |
| Ils / Elles | sont |

Pronunciation notes: Ils sont /il sɔ̃/ (іль сон). Elles sont /ɛl sɔ̃/ (ель сон).

Negative form: see [grammar/negation.md](../../../grammar/negation.md).

---

# Fixed Expressions *(if applicable)*

Être + profession/nationality takes **no article** (unlike English): "Je suis médecin", not "Je suis un médecin". See [grammar/profession-nationality-no-article.md](../../../grammar/profession-nationality-no-article.md).

---

# Examples

- Vous êtes japonais? — Oui, je suis japonais.
- Je suis médecin. Je suis chimiste. Je suis ukrainienne.
- Max est étudiant. Il est anglais. Léa est étudiante. Elle est anglaise.
- Nous sommes fatigué(e)s. Vous êtes fatigué(e)s.
- Ils sont italiens. Elles sont italiennes.
