# Basic Information

## French

savoir

## Part of Speech

Verb

## Ukrainian

знати / вміти

## English

to know (a fact) / to know how to

# Forms *(if applicable)*

## Verb Group

Group 3 (irregular)

---

# Conjugation *(if applicable)*

Present tense:

| Pronoun | Form |
|---|---|
| Je | sais |
| Tu | sais |
| Il / Elle / On | sait |
| Nous | savons |
| Vous | savez |
| Ils / Elles | savent |

See [grammar/connaitre-vs-savoir.md](../../../grammar/connaitre-vs-savoir.md) for the contrast with *connaître*.

---

# Examples

- Je sais parler français.
- Tu sais où est la gare.
- Nous savons lire. Vous savez chanter.
- Je ne sais pas ce que ce mot veut dire.
