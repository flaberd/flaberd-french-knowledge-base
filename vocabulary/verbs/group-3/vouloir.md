# Conjugation *(if applicable)*

Present tense:

| Pronoun | Form |
|---|---|
| Je | veux *(not "vais" — don't confuse with aller)* |
| Tu | veux |
| Il / Elle / On | veut |
| Nous | voulons |
| Vous | voulez |
| Ils / Elles | veulent |

The source flags a common mix-up in the nous/vous forms with [boire](boire.md): "Nous buvons, nous voulons" — don't confuse the two.
