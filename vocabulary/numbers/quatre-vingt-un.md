# Notes

Unlike the other ×1 compounds (21, 31, 41, 51, 61, 71 — which use "et"), 81 does **not** use "et": quatre-vingt-**un**, not quatre-vingt-et-un. See [grammar/numbers-80-and-above.md](../../grammar/numbers-80-and-above.md).
