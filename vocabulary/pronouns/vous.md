# Basic Information

## French

vous

## Part of Speech

Pronoun

## Ukrainian

ви

## English

you (plural, or formal singular)

---

# Examples

- Vous êtes fatigué(e)s.
- Vous êtes japonais?
- Je vous dis la réponse. *(à vous → vous, as an indirect object pronoun)*

---

# Notes

2nd person plural / polite subject pronoun. See [grammar/personal-pronouns.md](../../grammar/personal-pronouns.md). The same form "vous" is also used as the indirect object (COI) pronoun for "à vous" — see [grammar/indirect-object-pronouns.md](../../grammar/indirect-object-pronouns.md).
