# Basic Information

## French

les

## Part of Speech

Pronoun

## Ukrainian

їх (займенник прямого додатка, мн.)

## English

them (direct object, plural)

---

# Examples

- Je les aime. *(les chats → les)*
- Je les comprends. *(les livres → les)*
- Il regarde les étoiles. → Il les regarde.

---

# Notes

Direct object (COD) pronoun, plural (masculine or feminine). See [grammar/direct-object-pronouns.md](../../grammar/direct-object-pronouns.md).
