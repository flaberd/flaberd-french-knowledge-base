# Basic Information

## French

nous

## Part of Speech

Pronoun

## Ukrainian

ми

## English

we

---

# Examples

- Nous sommes fatigué(e)s.
- Nous sommes étudiants.
- Tu nous dis tout ? *(à nous → nous, as an indirect object pronoun)*

---

# Notes

1st person plural subject pronoun. See [grammar/personal-pronouns.md](../../grammar/personal-pronouns.md). The same form "nous" is also used as the indirect object (COI) pronoun for "à nous" — see [grammar/indirect-object-pronouns.md](../../grammar/indirect-object-pronouns.md).
