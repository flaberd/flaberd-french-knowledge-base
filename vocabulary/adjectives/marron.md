# Irregularities *(if applicable)*

*Marron* is a true invariable color: unlike most color adjectives, it never changes for gender **or** number (des chaussures marron, not marronnes). It is also a noun naming the chestnut. See [grammar/color-agreement.md](../../grammar/color-agreement.md).
