# Examples

- un garage jaune et vert.
- Des jupes __________. (jaune) *(color-agreement exercise: des jupes jaunes)*

---

# Notes

The feminine form is identical to the masculine simply because *jaune* already ends in unstressed -e (the regular pattern), not because it is a true invariable color like *orange*/*marron*. See [grammar/color-agreement.md](../../grammar/color-agreement.md).
