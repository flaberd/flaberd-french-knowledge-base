## 2. L'interrogation simple (-) : ne + pas

| Pronoun | Negated form |
|---|---|
| Je | Je ne suis pas |
| tu | Tu n'es pas |
| il / elle / on | Il n'est pas, Elle n'est pas, On n'est pas |
| nous | Nous ne sommes pas |
| vous | Vous n'êtes pas |
| ils/elles | Ils ne sont pas, Elles ne sont pas |

See [grammar/negation.md](../grammar/negation.md).

## 3. L'interrogation simple (?)

- **L'affirmation (+):** intonation descendante. "Vous êtes japonais." / "Jean est français."
- **L'interrogation (?):** intonation montante. "Vous êtes japonais ?" / "Jean est français ?"

### L'interrogation avec « est-ce que »

"Est-ce que vous êtes japonais ?" / "Est-ce que Jean est français ?"

"Est-ce que vous êtes monsieur Takashi ? Non, je ne suis pas monsieur Takashi."

Elision before il/elle/on/ils/elles (as given in source): Est-ce **qu'il**.. / Est-ce **qu'elle**.. / Est-ce **qu'on**.. / Est-ce **qu'ils**.. / Est-ce **qu'elles**..

See [grammar/interrogation.md](../grammar/interrogation.md).
