## 3. Le Passé Composé *(new grammar — major topic)*

> Le passé composé est un temps du passé utilisé pour exprimer une action ponctuelle, terminée, souvent brève ou ayant une conséquence dans le présent.
>
> Hier, j'ai mangé une pomme.
>
> Expressions de temps précises (indiquent une action passée terminée): Hier, la semaine dernière, le mois dernier, l'année dernière, il y a (deux jours, une heure, un an, etc.), ce matin (quand l'action est terminée), tout à l'heure, dès que, à ce moment-là, une fois, soudain, ensuite, puis, après.

## 5. Formation du passé composé *(new grammar)*

> avoir ou être au présent (l'auxiliaire) + le participe passé du verbe principal
>
> Hier, j'ai mangé une pomme. Elle est arrivée tard.

## 6. Choix de l'auxiliaire *(new grammar)*

> 1) La plupart – avoir. 2) Certains verbes – être : aller, venir, arriver, partir, entrer, sortir, monter, descendre, naître, mourir, rester, retourner, tomber, passer.
>
> Hier, j'ai bu du vin. Hier, je suis arrivé au Canada.

## 7. Participes passés *(new grammar)*

Six pattern groups, given as tables:

- **1er groupe, -er → -é**: parler → parlé, manger → mangé, rentrer → rentré.
- **2e groupe, -ir → -i**: finir → fini, choisir → choisi, réussir → réussi.
- **3e groupe, -u** *(irregular -oir verbs + the regular -re pattern, sharing the same ending)*: pouvoir → pu, vouloir → voulu, voir → vu, devoir → dû, savoir → su, recevoir → reçu, tenir → tenu, croire → cru, perdre → perdu, lire → lu, boire → bu, venir → venu, descendre → descendu, connaître → connu, vivre → vécu.
- **3e groupe, -is**: mettre → mis, prendre → pris, comprendre → compris, apprendre → appris, asseoir → assis.
- **3e groupe, -ert**: ouvrir → ouvert, offrir → offert, souffrir → souffert, découvrir → découvert.
- **3e groupe, -it**: conduire → conduit, construire → construit, produire → produit, traduire → traduit, dire → dit, écrire → écrit.
- **3e groupe, -i**: rire → ri, suivre → suivi.
- **Divers**: être → été, avoir → eu, faire → fait, naître → né, mourir → mort.
