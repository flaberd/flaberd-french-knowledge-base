## 4. « un », « une », « des » et « le », « la », « les »

- une tour → la tour Eiffel
- un pont → le Pont-Neuf
- des bateaux → les bateaux-mouches

**L'article indéfini** :

- Chose non unique, catégorie : un pont, une femme, des rues
- Quantité (un = 1) : un homme, une femme

**L'article défini** :

- Chose unique, précise : le Pont-Neuf, la femme de Paul, les rues de Paris
- Généralité : l'homme (= les hommes), la femme (= les femmes), la musique, le sport, le cinéma

Élision : le ami → l'ami ; le homme → l'homme.

See [grammar/definite-articles.md](../grammar/definite-articles.md) and [grammar/indefinite-articles.md](../grammar/indefinite-articles.md).

## 5. Gender, what's the trick? — terminaisons masculines

La finale indique parfois le genre.

Masculin :

- **-ment** : un gouvernement, un médicament...
- **-phone** : un téléphone, un interphone...
- **-scope** : un magnétoscope, un camescope...
- **-eau** : un bureau, un couteau...
- **-teur** : un aspirateur, un ordinateur...
- **-age** : un garage, un fromage... (exceptions : une image, une plage, une page...)

Note (source) : beaucoup de noms en « -e » sont masculins : le système, le problème, le programme, le modèle, le groupe, le sourire...

## 6. Gender, what's the trick? — terminaisons féminines

Féminin :

- **-tion / -sion** : une solution, une décision, une télévision...
- **-té** : la réalité, la société, la beauté...
- **-ure** : la culture, la peinture...
- **-ette** : une bicyclette, une disquette...
- **-ence / -ance** : la différence, la référence, la connaissance...

Note (source) : beaucoup de noms en « -eur » sont féminins : la fleur, la couleur, la peur, la valeur...

See [grammar/noun-gender-endings.md](../grammar/noun-gender-endings.md).
