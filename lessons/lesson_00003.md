## 2. Nouvelle règle : combinaisons de voyelles

New pronunciation rules for vowel combinations:

1. **"ill"** → [ij]: *famille* [famij]. Exceptions: *ville*, *mille*, *tranquille*, *Lille*.
2. **"oy"**, **"oi"** → [wa]: *trois* [trwa].
3. **"ai"**, **"ei"** → [ɛ]: *mais* [mɛ], *frais* [fʁɛ], *treize* [tʁɛz], *reine* [ʁɛn].
4. **"eau"**, **"au"** → [o]: *beaucoup* [boku], *auto* [oto].
5. **"eu"**, **"œu"**, **"e"** → [œ] / [ø] / [ə]: *neuf* [nœf], *pneu* [pnø], *regarder* [ʁəgarde] (teacher's note: practice with an online translator).
6. **"ou"** → [u].

## 3. Les sons nasaux (nasal sounds)

*Teacher's note: no equivalent in any language (as stated in the source).*

1. **"on"**, **"om"** → [ɔ̃]: *bon*, *monde*, *chanson*.
2. **"en"**, **"em"**, **"an"**, **"am"** → [ɑ̃]: *vent*, *temps*, *chambre*, *enfant*, *ensemble*.
3. **"in"**, **"im"**, **"ain"**, **"aim"**, **"ein"**, **"ym"**, **"yn"**, **"un"** → [ɛ̃]: *vin*, *simple*, *pain*, *faim*, *plein*, *un*, *matin*, *important*.
4. **"ien"** → [jɛ̃]: *chien*, *rien*, *canadien*.

Teacher's important note (given in Ukrainian in the source): if there is a doubled **nn**/**mm**, and/or a vowel follows, the nasal sound disappears and the pronunciation becomes the regular (non-nasal) one — e.g. *ukrainienne*, *bonne*.
