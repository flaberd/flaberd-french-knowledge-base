## 7. Dire — conjugaison *(new grammar + vocabulary)*

| Pronom | Conjugaison |
|---|---|
| Je | Je **dis** |
| tu | Tu **dis** |
| il / elle / on | Il **dit**, elle **dit**, on **dit** |
| nous | Nous **disons** |
| vous | Vous **dites** |
| ils/elles | Ils **disent**, Elles **disent** |

New verb entry: [vocabulary/verbs/group-3/dire.md](../vocabulary/verbs/group-3/dire.md). *Dire* was previously listed only as a 3e groupe classification example (see [lesson_00010](lesson_00010.md)); its full conjugation is taught here for the first time.

## 9. Venir (très irrégulier) — conjugaison *(new grammar + vocabulary)*

| Pronom | Conjugaison |
|---|---|
| Je | Je **viens** |
| tu | Tu **viens** |
| il / elle / on | Il **vient**, elle **vient**, on **vient** |
| nous | Nous **venons** |
| vous | Vous **venez** |
| ils/elles | Ils **viennent**, Elles **viennent** |

> Je viens **du** Canada. Tu viens **d'**Inde. Il vient **d'**Ukraine. Ils viennent **des** États-Unis.

The examples double as a demonstration of "venir de" + country, mirroring the au/aux/en contractions already documented for "aller à" + country (see [grammar/prepositions-of-place.md](../grammar/prepositions-of-place.md), now updated with a "coming from" section). New verb entry: [vocabulary/verbs/group-3/venir.md](../vocabulary/verbs/group-3/venir.md).

## 12. COD (complément d'objet direct) *(new grammar)*

| COD | Remplacé par… | Exemple |
|---|---|---|
| le fruit | le | Je **le** mange |
| la pomme | la | Tu **la** prends |
| Marie | la | Je **la** vois |
| les chats | les | Je **les** aime |
| les livres | les | Je **les** comprends |
| la nature | l' | Nous **l'**aimons |

New grammar file: [grammar/direct-object-pronouns.md](../grammar/direct-object-pronouns.md).

## 13. Les pronoms COI avec « dire » *(new grammar)*

| À + personne | Remplacé par… | Exemple |
|---|---|---|
| à moi | me | Il **me** dit bonjour |
| à toi | te | Je **te** dis la vérité |
| à lui / à elle | lui | Elle **lui** dit merci |
| à nous | nous | Tu **nous** dis tout ? |
| à vous | vous | Je **vous** dis la réponse |
| à eux / à elles | leur | Il **leur** dit bonjour |

New grammar file: [grammar/indirect-object-pronouns.md](../grammar/indirect-object-pronouns.md).

# Notes

- §1's corrected review sentences contain two source misspellings ("connessons"/"connessent" for connaissons/connaissent, "saves" for savez"), preserved verbatim rather than silently fixed.
- This lesson is split between a **recap** of connaître/savoir/devoir (§3–6, tables shown blank for active recall) and **substantial new material**: five new 3e groupe verb conjugations (dire, sortir, venir, voir, croire) and the first introduction of direct object (COD) and indirect object (COI) pronouns.
- §9's *venir* examples double as new content for prepositions of place ("venir de" + country); the au/aux/en/en(vowel) pattern already documented in [grammar/prepositions-of-place.md](../grammar/prepositions-of-place.md) for "aller à" is now paired with the "venir de" pattern.
- COD and COI are introduced only briefly here (one table each, illustrating "dire" for COI); they are developed much further in [lesson_00033](lesson_00033.md).
