*"Що помітили?"* (teacher's prompt — "What did you notice?")

## 3. Pluriel — rule

- **Pluriel = singulier + -s**: Il est blond → Ils sont blond**s**. Elle est blonde → Elles sont blond**es**. Il est étudiant → Ils sont étudiant**s**. Elle est étudiante → Elles sont étudiant**es**.
- **Important!** Finales en «-s» ou «-x»: pas de changement. Il est français. Ils sont français. Il est vieux. Ils sont vieux.
- **!!** Il est beau. Ils sont beau**x**. (applies to -eau, -au, or -eu endings)

### Les adjectifs ou noms masculins terminés par [-eau] [-au] ou [-eu]

- **Pluriel = singulier + -x**: Il est beau → Ils sont beaux. Un chapeau → Des chapeaux. Un jeu → Des jeux.

See [grammar/adjective-agreement-plural.md](../grammar/adjective-agreement-plural.md).
