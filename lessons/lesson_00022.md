## 1. Correction — texte descriptif (Barbara)

Graded text from the student's work, with the error marked in the source and corrected here:

> Barbara est très jeune : Elle a quinze ans. Elle est belle et elle est très intelligente. Elle a les yeux noirs et les cheveux blonds. Elle a beaucoup d'amis, parce qu'elle est très gentille. Elle a seulement un ~~grosse~~ **gros** défaut : elle est très paresseuse.

*(Error: "un grosse défaut" — "défaut" is masculine, so the adjective must be "gros", not the feminine "grosse". See [grammar/adjective-agreement-feminine.md](../grammar/adjective-agreement-feminine.md).)*

## 2. Correction — devoir sur les démonstratifs + position de l'adjectif (suite de lesson_00020, §11)

The student's graded answers to the "cette fille est jeune → c'est une jeune fille" exercise from [lesson_00020](lesson_00020.md), §11, as marked in the source:

> Cette fille est très jeune. Oui, c'est une jeune fille. *(correct, matches the model)*
> Ce manteau est nouveau. Oui, c'est un nouveau manteau. *(correct)*
> Ce manteau est nouveau. Oui, ce manteau est nouveau. *(repeated in the source as shown; does not use the "c'est un(e) + adj + nom" pattern the exercise was practicing)*
> Cet arbre est vieux. Non, cet arbre est jeune. *(does not use the "c'est un(e) + adj + nom" pattern; a closer answer would be "Non, c'est un jeune arbre")*
> Ces gâteaux sont très gros. Oui, ~~c'est des très gros gâteux~~. *(two errors: "c'est des" should be "**ce sont de**" — plural subject needs "ce sont", and "des" shifts to "de" before a preposed adjective; also "gâteux" is a misspelling of "gâteaux". See [grammar/adjective-position.md](../grammar/adjective-position.md).)*
> Cet appartement est nouveau. Oui, ~~c'est un nouveau appartment~~. *(two errors: "appartement" starts with a vowel, so "nouveau" must elide to "**nouvel**" — "c'est un nouvel appartement"; also "appartment" is a misspelling of "appartement". See [grammar/adjective-position.md](../grammar/adjective-position.md).)*
> Cet homme est très beau. Non, ce n'est pas un bel homme. *(correct — elision "bel" applied correctly)*
> Cette peinture est grosse. Oui, c'est une grosse peinture. *(correct)*
> Ces artistes sont nouveaux. Oui, ce sont de nouveaux artistes. *(correct — "ce sont de" used correctly here, unlike the gâteaux answer above)*
> Ce tableau est vieux. Oui, c'est un Vieux tableau. *(minor: "Vieux" capitalized mid-sentence — likely a typing slip, not a grammar error)*
> Ce chapeau est beau. Oui, c'est un beau chapeau *(correct; final period missing in source)*
> Cette création est nouvelle. Non, ce n'est pas une nouvelle creation. *(spelling: "creation" is missing its accent — should be "création")*

See [grammar/demonstrative-adjectives.md](../grammar/demonstrative-adjectives.md) and [grammar/adjective-position.md](../grammar/adjective-position.md).
