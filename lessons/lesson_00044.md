Corrections marked by the teacher:

| Flagged text | Issue |
|---|---|
| "Je suis **Ukrainien**" | Capitalization error: a nationality used as an adjective (after *être*) is lowercase in French — "je suis ukrainien". See [grammar/profession-nationality-no-article.md](../grammar/profession-nationality-no-article.md#common-mistake-capitalization). |
| "**Elle a** Il y a une rivière" | Correction shown inline: "Elle a" (using *avoir* for a city having a feature) is replaced with the more natural impersonal "**Il y a**". |
| "J'habite dans le centre-ville un appartement au **quinze** étage." | Two problems in one sentence: missing preposition structure, and a cardinal number ("quinze") used where an ordinal is needed for a floor ("quinzième"). Both are fixed in the essay's own closing line: "J'habite dans un appartement au **quinzième** étage, au centre-ville". |
| "Je **me brosse** et me lave" | Incomplete reflexive verb — *se brosser* alone is ambiguous; needs an object ("je me brosse les dents"). |
| "Je travaille avec **des gens bien**" | Flagged; the essay's own closing line supplies the fix: "**DE BONS GENS**" — when a plural noun is preceded by a preposed adjective, "des" becomes "de" (des gens + bons → de bons gens). New grammar: see [grammar/indefinite-articles.md](../grammar/indefinite-articles.md#des--de-before-a-preposed-adjective). |
| "je **vais** chez moi. (RENTRER!)" | Word choice: the teacher's bracketed note calls for *rentrer* (to go home/return) instead of *aller* + chez moi. |
| "je fais **de** promenade" | Wrong/missing article — should be "une promenade" or "des promenades" (this is not a negation or quantity context, so "de" alone is wrong here). |
| "Nous aimons **des** promenades" | Should be "**les** promenades" — *aimer* + noun always takes the definite article, never the partitive/indefinite. See [grammar/definite-articles.md](../grammar/definite-articles.md#with-verbs-of-opinionpreference). |
| "du thé avec **citron**" *(both occurrences)* | Missing article — likely "avec **du** citron" (partitive, indeterminate quantity), matching the pattern already used correctly elsewhere in the same essay ("du café", "du sucre"). |
| "Je veux acheter **la** voiture" | Definite article used where the car hasn't been mentioned before — should be indefinite "**une** voiture" (first mention, non-specific). |
| "Je veux **prendre** de vacance" | Flagged (verb highlighted in a third color, distinct from red/green — exact issue unclear beyond the adjacent article/number problem). "de vacance" should be "**des vacances**" (plural, correct partitive/indefinite form) — this half of the correction is clear; why *prendre* itself is highlighted is not stated in the source. |
