## 1. Les couleurs

**Une couleur** (fém.)

Cependant, certaines couleurs s'emploient comme noms masculins (le nom de la couleur elle-même) :

- le blanc, le bleu, le rouge, le noir

> Le bleu est ma couleur préférée.

### Accord des couleurs

- Un pantalon bleu → Une robe bleue
- Une robe blanche
- Une écharpe rouge *(rouge ne change pas au féminin, ex. jaune, jeune)*

### Couleurs invariables

Pourquoi dit-on *des vestes orange* (et non *oranges*) et *des chaussures marron* (et non *marronnes*) ?

En français, la plupart des adjectifs s'accordent en genre et en nombre avec le nom, mais certaines couleurs sont des exceptions : elles sont invariables.

Les couleurs invariables (qui ne changent jamais) : **orange**, **marron**.

*(Source note: "Ce n'est pas très important de retenir cela maintenant. Vous pouvez juste le reconnaître.")*

## 2. Groupe… c'est quoi ? (recap + 2e groupe)

| | 1er Groupe | 2e groupe | 3e groupe |
|---|---|---|---|
| Verbes en | -er | -ir | -ir\*, -oir, -re |
| Exemples | aimer, chanter, regarder, écouter, manger, parler… | finir, grandir, grossir, maigrir, réussir, choisir, agir… | dormir\*, offrir\*, venir, courir, savoir, vouloir, pouvoir, recevoir, prendre, dire, écrire, faire, lire, boire |
| Régularité | Conjugaison régulière. **Pas "aller"** (irrégulier, 3e groupe) | Conjugaison régulière | Groupe irrégulier avec plusieurs sous-types |

See [grammar/verb-groups.md](../grammar/verb-groups.md).

## 3. Les verbes du 2e groupe (en -ir) au présent

Modèle : **grandir**

| Pronom | Terminaison | Exemple |
|---|---|---|
| Je | -is | je grandis |
| tu | -is | tu grandis |
| il / elle / on | -it | il grandit |
| nous | -issons | nous grandissons |
| vous | -issez | vous grandissez |
| ils/elles | -issent | ils grandissent |

See [grammar/second-group-verbs-present.md](../grammar/second-group-verbs-present.md).

## 4. Astuce

Ces verbes se forment souvent sur un adjectif et expriment une transformation :

- Vieille → vieillir → Je vieillis → Vous vieillissez
- Rouge → rougir → Je rougis → Vous rougissez
- Grande → grandir → Je grandis → Vous grandissez
- Gros(se) → ??? *(left as a question in source — expected answer: grossir)*

Autres verbes fréquents : finir – choisir – guérir – applaudir – ralentir – se réunir.

# Notes

- This lesson teaches the regular present-tense conjugation of 2e groupe (-ir) verbs, using *grandir* as the model, and formalizes color agreement, including the true invariable colors *orange* and *marron*.
- *S'endormir* (homework item 1) is **not** a 2e groupe verb despite ending in -ir: it is built on *dormir*, an irregular 3e groupe verb (see [lesson_00016](lesson_00016.md) note and [grammar/verb-groups.md](../grammar/verb-groups.md)). It is recorded here as it appears in the source homework, but filed under group 3 in the vocabulary.
- All other new -ir verbs introduced (remplir, fournir, bâtir, investir, réfléchir, atterrir, applaudir, obéir, se réunir, se divertir, s'épanouir, se réjouir, se rafraîchir, s'accomplir, s'endurcir, s'investir) follow the regular 2e groupe pattern taught in this lesson.
