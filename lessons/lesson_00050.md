## 6. Ces verbes indiquent un changement de lieu *(new grammar — refinement)*

> arriver | partir → il est arrivé / il est parti. entrer | sortir → il est entré / il est sorti. monter | descendre | tomber → il est monté / il est descendu / il est tombé. aller | venir → il est allé / il est venu. naître | mourir → il est né / il est mort. passer → il est passé. retourner → il est retourné. rester → il est resté.
>
> En tout : **14 verbes** (+ les composés : rentrer, devenir, revenir, repartir...).

> * « Marcher », « courir », « sauter », etc. (mouvement simple), s'utilisent avec « avoir ».

## 7. Le participe passé s'accorde avec le sujet du verbe « être » *(new grammar)*

> je suis né / née, tu es né / née, il est né, elle est née, nous sommes nés / nées, vous êtes nés / nées, ils sont nés, elles sont nées.
>
> Il est arrivé / Elle est arrivée. Il est entré / Elle est entrée.
>
> Avoir : Ne s'accorde pas ! Elle a mangé. Il a mangé.

The first explicit teaching of être-subject agreement (previously only shown in passing via single examples). Updated [grammar/passe-compose.md](../grammar/passe-compose.md#participle-agreement-with-être) with the full table.

## 9. Suivis d'un complément d'objet, ces verbes se construisent avec « avoir » *(new grammar)*

> Sans complément : Elle est sortie. Elle est entrée. Elle est descendue.
> Avec un complément : Elle a sorti le chien. Elle a rentré la voiture. Elle a descendu la poubelle.

A key refinement: sortir, rentrer, descendre (and by extension monter, passer) switch to **avoir** when followed by a direct object — the verb becomes transitive ("to take/bring something in/out/down") rather than describing the subject's own motion. Added to [grammar/passe-compose.md](../grammar/passe-compose.md#exception-the-same-verb-can-take-avoir-instead-with-a-direct-object).

## 10. Les verbes pronominaux se construisent avec « être » *(new grammar)*

> Se lever (Inscrire) : Il a inscrit son frère, elle s'est inscrite hier.
>
> Je me suis levé(e), tu t'es levé(e), il s'est levé, elle s'est levée, on s'est levé(e)s, nous nous sommes levé(e)s, vous vous êtes levé(e)s, ils se sont levés, elles se sont levées.
>
> Je me lève x je me suis levé - Pourquoi?

Reflexive verbs always take être, with the pronoun placed before être and the participle agreeing with the subject — same mechanism as any other être-verb. The *inscrire*/*s'inscrire* pair is given as a direct contrast: non-reflexive + avoir (no agreement) vs. reflexive + être (agreement). The closing question ("why does je me lève become je me suis levé?") is rhetorical/unanswered in the source. New grammar added to [grammar/reflexive-verbs.md](../grammar/reflexive-verbs.md#passé-composé) and [grammar/passe-compose.md](../grammar/passe-compose.md#pronominal-reflexive-verbs-always-take-être). New vocabulary: [inscrire](../vocabulary/verbs/group-3/inscrire.md).
