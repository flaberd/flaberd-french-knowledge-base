## 3. La nationalité et la profession (être sans article)

- Vous êtes japonais? — Oui, je suis japonais.
- Vous êtes médecin? — Non, je suis chimiste.
- **Правильно:** Je suis médecin. Je suis chimiste. Je suis ukrainienne.
- **Не правильно:** Je suis un médecin. Je suis un chimiste. Je suis une ukrainienne.

See [grammar/profession-nationality-no-article.md](../grammar/profession-nationality-no-article.md).

## 4. La description (accord masculin/féminin)

Max est étudiant. Il est anglais. Il est marié. Il est grand. Il est blond.
Léa est étudiante. Elle est anglaise. Elle est mariée. Elle est grande. Elle est blonde.

Max: Il est italien. Il est lycéen.
Léa: Elle est italienne. Elle est lycéenne.

Max: Il est jeune.
Léa: Elle est jeune.

*"Що помітили?"* (teacher's prompt — "What did you notice?")

### Rule given in source

Féminin = masculin + **-e** : étudiant → étudiant**e**
-en devient -enne : italien → itali**enne**

Pronunciation note (as given): "Il est_anglais. Il est_amusant" — read with a liaison (linking) to a [t] sound when followed by a vowel.

See [grammar/adjective-agreement-feminine.md](../grammar/adjective-agreement-feminine.md).
