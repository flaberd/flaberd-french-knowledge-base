## 4. Avoir

| Pronom | Conjugaison |
|---|---|
| Je | J'**ai** |
| tu | Tu **as** |
| il / elle / on | Il **a**, elle **a**, on **a** |
| nous | Nous **avons** |
| vous | Vous **avez** |
| ils/elles | Ils **ont**, Elles **ont** |

\*Ne confondez pas "Ils sont" et "Elles sont" (verbe être) avec "Ils ont"/"Elles ont" (verbe avoir).

Prononciation (liaison), d'après le schéma de la source :

- On_a [pas de liaison marquée]
- Nous_avons [z]
- Vous_avez [z]
- Ils_ont [z]
- Ils_sont [s] *(contraste avec "Ils ont" [z])*

## 6. La place de l'adjectif en français

En général, l'adjectif se place **après** le nom :

- Un livre intéressant
- Une maison moderne
- Des étudiants intelligents

! Certains adjectifs se placent **avant** le nom :

| Adjectif | Exemple |
|---|---|
| beau, joli | un beau paysage |
| jeune, vieux | une jeune femme |
| premier, deux | le premier jour, trois chats |
| bon, mauvais | un bon élève, une mauvaise idée |
| grand, petit | un grand homme, une petite fille |

### Moment — élision et « des » → « de »

Devant une voyelle ou un h muet (masculin) :

- beau → **bel** : un bel acteur
- vieux → **vieil** : un vieil homme
- nouveau → **nouvel** : un nouvel ordinateur

Lorsque l'adjectif est placé **devant** le nom, l'article défini pluriel "des" se transforme en "de" (ou "d'" devant voyelle) :

- J'ai **de** beaux livres. Elle a **de** grandes idées.
- *(Incorrect)* J'ai ~~des beaux livres~~. Elle a ~~des grandes idées~~.
- J'ai **des** livres intéressants. *(adjectif après le nom → "des" reste "des")*

See [grammar/adjective-position.md](../grammar/adjective-position.md).

> Dites : Vous avez chaud ?

Petit rappel (à retenir !) :

- à + le = au
- à + les = aux
- à + la/l' = à la/l'
