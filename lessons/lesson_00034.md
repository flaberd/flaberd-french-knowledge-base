## 1. La révision — corrections

> Elle prépare le repas dans la cuisine. Elle **lui** prépare dans la cuisine.
>
> Il écoute la chanson. → Il **la** écoute. (cod)
>
> **Écrire** (conjugaison!)

These appear to be corrected/reviewed answers to [lesson_00033](lesson_00033.md)'s homework. Two issues are preserved verbatim rather than silently fixed:

- "Elle **lui** prépare dans la cuisine" uses the COI pronoun **lui**, but "le repas" is a direct object (COD) — the expected pronoun would be **le** ("Elle le prépare dans la cuisine"). This may carry over the (already-noted) header/answer mismatch from [lesson_00033](lesson_00033.md) §16.
- "Il **la** écoute" is missing the elision before a vowel — it should be "Il **l'**écoute" (la → l' before a vowel sound, same rule as le/la → l'; see [grammar/direct-object-pronouns.md](../grammar/direct-object-pronouns.md)).

"Écrire (conjugaison!)" flags that *écrire*'s conjugation is coming up in a future lesson; no table is given here. [vocabulary/verbs/group-3/écrire.md](../vocabulary/verbs/group-3/écrire.md) still records its conjugation as "not yet taught."
