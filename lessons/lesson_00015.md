## 2. Un homme, une femme / un arbre, une route — l'article indéfini

"Un" et "une" sont des articles indéfinis utilisés pour désigner une chose ou une personne non spécifiée ou mentionnée pour la première fois.

- Masculin — personne : **un** homme ; chose : **un** arbre
- Féminin — personne : **une** femme ; chose : **une** route

English: *a man*.

See [grammar/indefinite-articles.md](../grammar/indefinite-articles.md).

## 3. Masculin et féminin des noms

Féminin des noms de personnes = féminin des adjectifs :

- un étudiant élégant → une étudiante élégante
- un musicien italien → une musicienne italienne
- un serveur rêveur → une serveuse rêveuse
- un acteur séducteur → une actrice séductrice
- un journaliste triste → une journaliste triste

Cas particuliers : un homme/une femme, un monsieur/une dame, un garçon/une fille.

Prononciation : "un" + voyelle → liaison en [n] (*un ami*, *un homme*).

See [grammar/gender-of-nouns.md](../grammar/gender-of-nouns.md).

## 7. Des arbres, des fleurs, des oiseaux — le pluriel des noms

- un arbre → des arbres
- une fleur → des fleurs
- un oiseau → des oiseaux

Prononciation : liaison "des" + voyelle → son [z] (*des amis*, *des hommes*).

Règles du pluriel :

1. Pluriel des noms = singulier + **-s** : un garçon → des garçons ; une fille → des filles.
2. Finales en **-s**, **-x** ou **-z** : pas de changement : un pays → des pays ; un choix → des choix ; un gaz → des gaz.
3. Finales en **-al** ou **-au** → **-aux** : un journal → des journaux ; un gâteau → des gâteaux.

Remarque importante (source) : il y aura toujours des exceptions aux règles de formation du pluriel. Par exemple : bancal → bancals (et non bancaux), natal → natals. Il n'est pas nécessaire de se focaliser sur ces cas particuliers ; ils peuvent être assimilés progressivement.

See [grammar/adjective-agreement-plural.md](../grammar/adjective-agreement-plural.md).

## 8. Le pluriel des adjectifs en -al

Le pluriel masculin change de [-al] à [-aux] : local → locaux, locale → locales ; social → sociaux, sociale → sociales ; original → originaux, originale → originales.

Exemples : un projet original → des projets originaux ; un devoir moral → des devoirs moraux ; un problème social → des problèmes sociaux ; un travail local → des travaux locaux.

- Irregular plural: travail → travaux, vitrail → vitraux, bail → baux, émail → émaux, corail → coraux, soupirail → soupiraux, vantail → vantaux.
- Regular: détail → détails, chandail → chandails, portail → portails, rail → rails.
- Exceptions: banal → banals, fatal → fatals, natal → natals, final → finals, glacial → glacials.
