## 3. Masculin et féminin (suite)

New feminine-formation patterns given in the source:

- **-on → -onne**: Il est bon. Elle est bonne. / Il est breton. Elle est bretonne. / Il est mignon. Elle est mignonne.
- **-er → -ère**: Il est étranger. Elle est étrangère. / Il est boulanger. Elle est boulangère. / Il est infirmier. Elle est infirmière.
- **-eur/-eux → -euse**: Il est serveur. Elle est serveuse. / Il est heureux. Elle est heureuse.
- **-teur → -trice**: Il est acteur. Elle est actrice. / Il est éditeur. Elle est éditrice. / Il est agriculteur. Elle est agricultrice.
- ***Exception** (source's own label): chanteur/chanteuse, menteur/menteuse — these follow -eur→-euse rather than -teur→-trice.
