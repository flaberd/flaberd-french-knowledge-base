# Adverbs of frequency (les adverbes de fréquence)

Presented in the source as a scale from most to least frequent:

| French | English | Frequency |
|---|---|---|
| toujours | always | highest |
| tout le temps | all the time | |
| tous les jours | everyday | |
| souvent | often | |
| de temps en temps | from time to time | |
| parfois | sometimes | |
| quelquefois | sometimes | |
| rarement | rarely | |
| jamais | never | lowest |

*Parfois* and *quelquefois* are both glossed "sometimes" and sit at the same point on the scale.

## Notes

- Source slide is attributed to "L'Accent Français" (external reference material reproduced in the lesson).
- *Jamais* is also used in the negative construction **ne...jamais**; see [negation.md](negation.md).
