# Definite articles: le, la, les, l'

## Forms

| | Singular | Plural |
|---|---|---|
| Masculine | le (l' before a vowel) | les |
| Feminine | la (l' before a vowel) | les |

Elision: le ami → **l'ami** ; le homme → **l'homme** (the silent "h" behaves like a vowel here).

## Use

- **Unique / specific thing**: le Pont-Neuf, la femme de Paul, les rues de Paris.
- **Generality** (a category as a whole): l'homme (= les hommes, "man" in general), la femme (= les femmes), la musique, le sport, le cinéma.

Contrast with the indefinite article, used for a non-unique thing/category or a quantity of one — see [grammar/indefinite-articles.md](indefinite-articles.md).

## Indefinite vs. definite side by side

- une tour → **la tour Eiffel** (a tower → *the* Eiffel Tower)
- un pont → **le Pont-Neuf**
- des bateaux → **les bateaux-mouches**
