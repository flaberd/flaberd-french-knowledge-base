# Combinaisons de voyelles (vowel combinations)

1. **"ill"** → **[ij]**: *famille* [fa.mij].
   - Exceptions (pronounced **[il]**, not [ij]): *ville*, *mille*, *tranquille*, *Lille*.
2. **"oi"** / **"oy"** → **[wa]**: *trois* [tʁwa].
3. **"ai"**, **"ei"** → **[ɛ]**: *mais* [mɛ], *frais* [fʁɛ], *treize* [tʁɛz], *reine* [ʁɛn].
4. **"eau"**, **"au"** → **[o]**: *beaucoup* [boku], *auto* [oto].
5. **"eu"**, **"œu"**, **"e"** → **[œ]** / **[ø]** / **[ə]**: *neuf* [nœf], *pneu* [pnø], *regarder* [ʁə.gaʁ.de].
6. **"ou"** → **[u]**.
