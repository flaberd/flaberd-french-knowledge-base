# Adjective agreement: feminine forms

## Base rule

Féminin = masculin + **-e** : étudiant → étudiant**e**

## Common ending patterns

| Masculine ending | Feminine ending | Examples |
|---|---|---|
| (regular) | + -e | étudiant → étudiante, marié → mariée, grand → grande, blond → blonde, petit → petite, intelligent → intelligente |
| -en | -enne | italien → italienne, lycéen → lycéenne |
| -on | -onne | bon → bonne, breton → bretonne, mignon → mignonne |
| -er | -ère | étranger → étrangère, boulanger → boulangère, infirmier → infirmière |
| -eur / -eux | -euse | serveur → serveuse, heureux → heureuse, danseur → danseuse, travailleur → travailleuse |
| -teur | -trice | acteur → actrice, éditeur → éditrice, agriculteur → agricultrice, présentateur → présentatrice, instituteur → institutrice |
| -f | -ve | sportif → sportive, actif → active, créatif → créative |
| -et | -ette | cadet → cadette, coquet → coquette, net → nette |
| -el | -elle | habituel → habituelle, essentiel → essentielle, naturel → naturelle |

**Exception to -teur → -trice:** chanteur → chanteuse, menteur → menteuse (these follow the -eur → -euse pattern instead).

| blanc | blanche | білий → біла |
| frais | fraîche | свіжий → свіжа |
| long | longue | довгий → довга |
| fou | folle | божевільний → божевільна |
| doux | douce | ніжний → ніжна |
| faux | fausse | фальшивий → фальшива |

## Pronunciation note

When an adjective ending in a normally-silent consonant is followed by a vowel-initial word, that consonant is pronounced via liaison: "Il est_anglais", "Il est_amusant" (linked with a [t] sound).
