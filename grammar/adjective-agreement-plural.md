# Adjective (and noun) agreement: plural forms

## Base rule

Pluriel = singulier + **-s** : Il est blond → Ils sont blond**s**. Elle est blonde → Elles sont blond**es**. Il est étudiant → Ils sont étudiant**s**. Elle est étudiante → Elles sont étudiant**es**.

## Important exceptions

- **Endings in -s, -x, or -z: no change.** Il est français → Ils sont français. Il est vieux → Ils sont vieux. Un pays → des pays. Un choix → des choix. Un gaz → des gaz.
- **Endings in -eau, -au, or -eu: plural = singulier + -x.** Il est beau → Ils sont beaux. Un chapeau → Des chapeaux. Un jeu → Des jeux. Un gâteau → des gâteaux.
- **Endings in -al: plural → -aux** (masculine only; the feminine follows the regular +s rule). Un journal → des journaux. local → locaux / locale → locales. social → sociaux / sociale → sociales. original → originaux / originale → originales.
  - Examples: un projet original → des projets originaux ; un devoir moral → des devoirs moraux ; un problème social → des problèmes sociaux ; un travail local → des travaux locaux.
  - Irregular -ail nouns that also take -aux: travail → travaux, vitrail → vitraux, bail → baux, émail → émaux, corail → coraux, soupirail → soupiraux, vantail → vantaux.
  - Regular -ail nouns (just +s): détail → détails, chandail → chandails, portail → portails, rail → rails.
  - Exceptions to -al → -aux (just +s): banal → banals, fatal → fatals, natal → natals, final → finals, glacial → glacials, bancal → bancals.

There will always be exceptions to plural-formation rules; it is not necessary to memorize every particular case — they are absorbed gradually with exposure.
