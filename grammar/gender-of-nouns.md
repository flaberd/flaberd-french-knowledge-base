# Gender of nouns: feminine of person nouns

## Base rule

For nouns referring to people, the feminine form follows the same patterns as feminine adjectives: féminin = masculin + **-e**, with the same ending changes. See [grammar/adjective-agreement-feminine.md](adjective-agreement-feminine.md) for the full pattern table (-en→-enne, -eur/-eux→-euse, -teur→-trice, -er→-ère, etc.).

Examples from the source:

- un étudiant élégant → une étudiante élégante
- un musicien italien → une musicienne italienne
- un serveur rêveur → une serveuse rêveuse
- un acteur séducteur → une actrice séductrice
- un journaliste triste → une journaliste triste (no change: already ends in -e)

## Irregular / suppletive pairs (cas particuliers)

These do not follow a derivational pattern — the masculine and feminine forms are different words:

- un homme / une femme (man / woman)
- un monsieur / une dame (gentleman / lady)
- un garçon / une fille (boy / girl)

## Pronunciation

"un" + vowel-initial word → liaison in [n]: *un ami*, *un homme*.
