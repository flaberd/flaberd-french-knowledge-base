# Connaître vs. savoir

Both mean "to know" in English, but they are not interchangeable.

## Conjugation — connaître

| Pronom | Conjugaison |
|---|---|
| Je | Je **connais** |
| tu | Tu **connais** |
| il / elle / on | Il **connaît**, elle **connaît**, on **connaît** |
| nous | Nous **connaissons** |
| vous | Vous **connaissez** |
| ils/elles | Ils **connaissent**, Elles **connaissent** |

## Conjugation — savoir

| Pronom | Conjugaison |
|---|---|
| Je | Je **sais** |
| tu | Tu **sais** |
| il / elle / on | Il **sait**, elle **sait**, on **sait** |
| nous | Nous **savons** |
| vous | Vous **savez** |
| ils/elles | Ils **savent**, Elles **savent** |

## Usage — connaître

Used for:

- **Personnes**: *Je connais Paul.*
- **Lieux**: *Nous connaissons Montréal.*
- **Noms.**

Example: *Je ne connais pas ce mot en français.*

## Usage — savoir

Used for:

- **Faits / informations précises**: *Je sais que Marie est à Paris.*
- **Savoir-faire / compétences**: *Je sais nager.*
- **Phrases avec "que" ou "si"**: *Tu sais si le magasin est ouvert ?*

❌ **Not** for people or objects one has encountered/met.

Example: *Je ne sais pas ce que ce mot veut dire.*

## Quick contrast

| | connaître | savoir |
|---|---|---|
| People | ✅ (*Je connais Paul.*) | ❌ |
| Places | ✅ (*Nous connaissons Montréal.*) | ❌ |
| Names | ✅ | ❌ |
| Facts / information | ❌ | ✅ (*Je sais que...*) |
| Skills / know-how | ❌ | ✅ (*Je sais nager.*) |
| Clauses with "que"/"si" | ❌ | ✅ |

## Mnemonic

> **Connaître + Noms** (connaître + nouns — people, places, things)
> **Savoir + Que/si/où / verbe** (savoir + a clause or an infinitive)
