# Faire: present tense (irregular 3e groupe)

| Pronom | Conjugaison |
|---|---|
| Je | Je **fais** |
| tu | Tu **fais** |
| il / elle / on | Il **fait**, elle **fait**, on **fait** |
| nous | Nous **faisons** |
| vous | Vous **faites** |
| ils/elles | Ils **font**, Elles **font** |

## Notes

- *Faire* was already listed as a 3e groupe verb example (see [verb-groups.md](verb-groups.md)), but its own conjugation is taught here for the first time.
- *Vous faites* and *ils/elles font* are irregular, not built on the *fais-* stem.
- *Faire* is the base of the impersonal weather expressions — see [weather-expressions.md](weather-expressions.md).
