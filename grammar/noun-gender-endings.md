# Gender of nouns: predicting gender from the ending

Unlike person nouns (see [grammar/gender-of-nouns.md](gender-of-nouns.md)), the gender of thing-nouns is not derived from a masculine/feminine pair — but certain endings reliably predict it.

## Masculine endings

| Ending | Examples |
|---|---|
| -ment | un gouvernement, un médicament |
| -phone | un téléphone, un interphone |
| -scope | un magnétoscope, un camescope |
| -eau | un bureau, un couteau |
| -teur | un aspirateur, un ordinateur |
| -age | un garage, un fromage (exceptions: une image, une plage, une page) |

Note: many nouns ending in **-e** are masculine: le système, le problème, le programme, le modèle, le groupe, le sourire.

## Feminine endings

| Ending | Examples |
|---|---|
| -tion / -sion | une solution, une décision, une télévision |
| -té | la réalité, la société, la beauté |
| -ure | la culture, la peinture |
| -ette | une bicyclette, une disquette |
| -ence / -ance | la différence, la référence, la connaissance |

Note: many nouns ending in **-eur** are feminine: la fleur, la couleur, la peur, la valeur.
