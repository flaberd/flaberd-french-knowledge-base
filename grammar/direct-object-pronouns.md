# Direct object pronouns (COD — complément d'objet direct)

## What counts as a COD

A **COD** answers the question **"qui ?"** (who) or **"quoi ?"** (what) asked directly after the verb, and is **never introduced by a preposition**.

- *Le chat mange* **la souris**. → mange **quoi ?** → la souris (COD).
- *Arthur cherche* **son frère**. → cherche **qui ?** → son frère (COD).

## Pronoun table

| COD (noun) | Replaced by… |
|---|---|
| le fruit (masc. singular) | **le** |
| la pomme (fem. singular) | **la** |
| Marie (a person) | **la** |
| les chats / les livres (plural) | **les** |
| la nature (before a vowel sound) | **l'** |

**le** and **la** both elide to **l'** before a vowel or mute h.

## Examples

- Je le mange. Tu la prends. Je la vois. Je les aime. Je les comprends. Nous l'aimons.
- Je vois le ciel. → Je **le** vois.
- Tu prends la pomme. → Tu **la** prends.
- Il regarde les étoiles. → Il **les** regarde.
- Nous aimons le chocolat. → Nous **l'**aimons.
- Vous écoutez la musique. → Vous **l'**écoutez.

## Word order

The COD pronoun goes directly before the conjugated verb (not after, as in English): *Je le mange*, not ~~Je mange le~~.
