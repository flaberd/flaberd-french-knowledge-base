# Les lettres finales muettes (silent final letters)

## Rule

The following letters are usually **silent** when they appear at the end of a French word:

**E, D, S, X, Z, P, T, G**

This also applies to combinations like *es, ts, ds, ps*.

### Exception

**-é** is always pronounced (e.g. *la santé* → [san.te]).

### Mnemonic

- English: *Every Dog Sits, Xylophones Play Zumba, Time Goes*
- Ukrainian (teacher's version): *(Е)тот (Д)ед (С)пит, (Х)отя (З)вон (П)угает (Т)русливых (Г)усей*

## The letter H

**H** is never pronounced anywhere in French. It is only used to form other sounds/digraphs (e.g. *ph*, *ch*).

## Final "-e" reactivates the preceding consonant

Key rule: when a word ends in a silent consonant and a feminine **-e** is added, the consonant is no longer silent — the **-e** "reactivates" (makes pronounced) the consonant before it.

- *petit* [pə.ti] → *petite* [pə.tit]
- *grand* [ɡʁɑ̃] → *grande* [ɡʁɑ̃d]

## Practice words

climat, amie, trop, nez, ami, cadets, avec, sud, étudiant, étudiante, petit, petite
