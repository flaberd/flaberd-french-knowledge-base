# L'alphabet français (the French alphabet)

The French alphabet has the same 26 letters as English, but each letter name is pronounced differently.

| Letter | IPA        | Ukrainian approx. | Example word |
| ------ | ---------- | ----------------- | ------------ |
| A      | [a]        | [а]               | un ami       |
| B      | [be]       | [бэ]              | un bébé      |
| C      | [se]       | [сэ]              | un cinéma    |
| D      | [de]       | [дэ]              | une danse    |
| E      | [ø]        | [ё]               | un éléphant  |
| F      | [ɛf]       | [эф]              | une fleur    |
| G      | [ʒe]       | [жэ]              | un genou     |
| H      | [aʃ]       | [аш]              | un hibou     |
| I      | [i]        | [и]               | une île      |
| J      | [ʒi]       | [жи]              | le jus       |
| K      | [ka]       | [ка]              | un kangourou |
| L      | [ɛl]       | [эль]             | un lapin     |
| M      | [ɛm]       | [эм]              | une maison   |
| N      | [ɛn]       | [эн]              | un nid       |
| O      | [o]        | [о]               | l'orange     |
| P      | [pe]       | [пэ]              | une pomme    |
| Q      | [ky]       | [кю]              | quatre       |
| S      | [ɛs]       | [эс]              | une souris   |
| T      | [te]       | [тэ]              | une tortue   |
| U      | [y]        | [ю]               | un uniforme  |
| V      | [ve]       | [вэ]              | une vache    |
| W      | [dublə ve] | [дубль во]        | un wagon     |
| X      | [iks]      | [икс]             | un xylophone |
| R      | [ɛʁ]       | [эр]              | le raisin    |
| Y      | [igrɛk]    | [игрек]           | un yoyo      |
| Z      | [zɛd]      | [зэд]             | un zèbre     |

- CHAT
- ANATOLII
- CHIEN
