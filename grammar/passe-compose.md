# Le passé composé (compound past)

## Usage

Le passé composé is used for a **punctual, completed action**, often brief, or one with a consequence in the present. Contrast with [l'imparfait](imparfait.md), which describes ongoing states, habits, or background context rather than a finished, precise action.

- Hier, j'**ai mangé** une pomme.

## Time expressions that signal the passé composé

Hier, la semaine dernière, le mois dernier, l'année dernière, il y a (deux jours, une heure, un an, etc.), ce matin *(when the action is finished)*, tout à l'heure, dès que, à ce moment-là, une fois, soudain, ensuite, puis, après.

These contrast with [l'imparfait](imparfait.md#usage)'s signal words (souvent, tous les jours, toujours, d'habitude...) — precise/completed vs. habitual/ongoing.

## Formation

> Sujet + **avoir** ou **être** au présent (l'auxiliaire) + le **participe passé** du verbe principal

- Hier, j'**ai mangé** une pomme.
- Elle **est arrivée** tard.

## Choice of auxiliary

1. **Most verbs → avoir**: Hier, j'**ai** bu du vin.
2. **A specific set of verbs → être**: aller, venir, arriver, partir, entrer, sortir, monter, descendre, naître, mourir, rester, retourner, tomber, passer — exactly **14 verbs**, plus their compounds (rentrer, devenir, revenir, repartir...). Hier, je **suis** arrivé au Canada.

Mnemonic given in the source: **"la maison d'être"** — a house diagram where these verbs trace a path in and out of a house (monter/descendre the stairs, naître/mourir inside, arriver/partir, entrer/sortir, tomber off the roof, (re)venir/aller at the door, retourner looping underneath). All of these describe **a change of place or state**.

This 14-verb list is closed and memorized — other simple-movement verbs (marcher, courir, sauter...) still take **avoir**: il **a** couru, il **a** marché.

### Exception: the same verb can take avoir instead, with a direct object

A handful of these être-verbs (sortir, rentrer, descendre, monter, passer) switch to **avoir** when used **transitively** (followed by a direct object) — the meaning shifts from "to go in/out/down" to "to take/bring something in/out/down":

| Sans complément (être) | Avec un complément (avoir) |
|---|---|
| Elle est sortie. | Elle **a sorti** le chien. |
| Elle est rentrée. | Elle **a rentré** la voiture. |
| Elle est descendue. | Elle **a descendu** la poubelle. |

## Participle agreement with être

With être as the auxiliary, the past participle **agrees in gender/number with the subject** (like an adjective):

| Sujet | Forme |
|---|---|
| je (m/f) | je suis né / née |
| tu (m/f) | tu es né / née |
| il | il est né |
| elle | elle est née |
| nous (m/f) | nous sommes nés / nées |
| vous (m/f) | vous êtes nés / nées |
| ils | ils sont nés |
| elles | elles sont nées |

Same pattern for any être-verb: il est arrivé / elle est arriv**ée**.

**With avoir, the participle normally does not agree** with the subject: elle **a mangé**, il **a mangé** (no -e/-s added, regardless of who's speaking). See [grammar/direct-object-pronouns.md](direct-object-pronouns.md#participle-agreement-in-the-passé-composé) for the one case where an avoir-participle *does* agree — with a preceding direct object, not the subject.

## Pronominal (reflexive) verbs always take être

Reflexive verbs (se lever, s'habiller, s'endormir...) always use **être** in the passé composé, with the reflexive pronoun placed before être, and the participle agreeing with the subject like any other être-verb:

| Sujet | SE LEVER (passé composé) |
|---|---|
| je | je me suis levé(e) |
| tu | tu t'es levé(e) |
| il | il s'est levé |
| elle | elle s'est levée |
| on | on s'est levé(e)s |
| nous | nous nous sommes levé(e)s |
| vous | vous vous êtes levé(e)s |
| ils | ils se sont levés |
| elles | elles se sont levées |

Contrast with the same underlying verb used non-reflexively with avoir: *inscrire* — "il **a inscrit** son frère" (avoir, non-reflexive, direct object "son frère") vs. "elle **s'est inscrite**" (être, reflexive — *s'inscrire*, "to enroll (oneself)"). See [grammar/reflexive-verbs.md](reflexive-verbs.md#passé-composé) for the full pattern.

## Formation of the past participle

### 1er groupe: -er → -é

parler → parl**é**, manger → mang**é**, rentrer → rentr**é**.

### 2e groupe: -ir → -i

finir → fin**i**, choisir → chois**i**, réussir → réuss**i**.

### 3e groupe: irregular patterns

**-u** *(includes genuinely irregular -oir verbs and the regular -re pattern, grouped together here since they share the same ending)*:

| Verb | Participe passé |
|---|---|
| pouvoir | pu |
| vouloir | voulu |
| voir | vu |
| devoir | dû |
| savoir | su |
| recevoir | reçu |
| tenir | tenu |
| croire | cru |
| perdre | perdu |
| lire | lu |
| boire | bu |
| venir | venu |
| descendre | descendu |
| connaître | connu |
| vivre | vécu |

**-is**: mettre → mis, prendre → pris, comprendre → compris, apprendre → appris, asseoir → assis.

**-ert**: ouvrir → ouvert, offrir → offert, souffrir → souffert, découvrir → découvert.

**-it**: conduire → conduit, construire → construit, produire → produit, traduire → traduit, dire → dit, écrire → écrit.

**-i**: rire → ri, suivre → suivi.

**Divers** (no shared pattern): être → été, avoir → eu, faire → fait, naître → né, mourir → mort.

## Examples with avoir + 1er groupe

- Il achète du pain. → Il **a acheté** du pain.
- Elle mange une pomme. → Elle **a mangé** une pomme.
- En 1995, j'**ai gagné** au Loto. L'année dernière, j'**ai acheté** une voiture.
