# Le passé composé (compound past)

## Usage

Le passé composé is used for a **punctual, completed action**, often brief, or one with a consequence in the present. Contrast with [l'imparfait](imparfait.md), which describes ongoing states, habits, or background context rather than a finished, precise action.

- Hier, j'**ai mangé** une pomme.

## Time expressions that signal the passé composé

Hier, la semaine dernière, le mois dernier, l'année dernière, il y a (deux jours, une heure, un an, etc.), ce matin *(when the action is finished)*, tout à l'heure, dès que, à ce moment-là, une fois, soudain, ensuite, puis, après.

These contrast with [l'imparfait](imparfait.md#usage)'s signal words (souvent, tous les jours, toujours, d'habitude...) — precise/completed vs. habitual/ongoing.

## Formation

> Sujet + **avoir** ou **être** au présent (l'auxiliaire) + le **participe passé** du verbe principal

- Hier, j'**ai mangé** une pomme.
- Elle **est arrivée** tard.

## Choice of auxiliary

1. **Most verbs → avoir**: Hier, j'**ai** bu du vin.
2. **A specific set of verbs → être**: aller, venir, arriver, partir, entrer, sortir, monter, descendre, naître, mourir, rester, retourner, tomber, passer (+ their compounds, e.g. rentrer, revenir). Hier, je **suis** arrivé au Canada.

Mnemonic given in the source: **"la maison d'être"** — a house diagram where these verbs trace a path in and out of a house (monter/descendre the stairs, naître/mourir inside, arriver/partir, entrer/sortir, tomber off the roof, (re)venir/aller at the door, retourner looping underneath). All of these describe **motion or a change of state/place** without a direct object.

With être as the auxiliary, the past participle **agrees in gender/number with the subject** (like an adjective) — e.g. elle est arriv**ée**, ils sont all**és**, elles sont parti**es**. *(Agreement mechanics beyond the single example given in the source are not detailed here; flagged for a future lesson.)*

Reflexive/pronominal verbs also take être (not covered explicitly in this lesson's slides, but implied by "s'endormir", "se brosser" etc. already in the vocabulary — cross-reference [grammar/reflexive-verbs.md](reflexive-verbs.md)).

## Formation of the past participle

### 1er groupe: -er → -é

parler → parl**é**, manger → mang**é**, rentrer → rentr**é**.

### 2e groupe: -ir → -i

finir → fin**i**, choisir → chois**i**, réussir → réuss**i**.

### 3e groupe: irregular patterns

**-u** *(includes genuinely irregular -oir verbs and the regular -re pattern, grouped together here since they share the same ending)*:

| Verb | Participe passé |
|---|---|
| pouvoir | pu |
| vouloir | voulu |
| voir | vu |
| devoir | dû |
| savoir | su |
| recevoir | reçu |
| tenir | tenu |
| croire | cru |
| perdre | perdu |
| lire | lu |
| boire | bu |
| venir | venu |
| descendre | descendu |
| connaître | connu |
| vivre | vécu |

**-is**: mettre → mis, prendre → pris, comprendre → compris, apprendre → appris, asseoir → assis.

**-ert**: ouvrir → ouvert, offrir → offert, souffrir → souffert, découvrir → découvert.

**-it**: conduire → conduit, construire → construit, produire → produit, traduire → traduit, dire → dit, écrire → écrit.

**-i**: rire → ri, suivre → suivi.

**Divers** (no shared pattern): être → été, avoir → eu, faire → fait, naître → né, mourir → mort.

## Examples with avoir + 1er groupe

- Il achète du pain. → Il **a acheté** du pain.
- Elle mange une pomme. → Elle **a mangé** une pomme.
- En 1995, j'**ai gagné** au Loto. L'année dernière, j'**ai acheté** une voiture.
