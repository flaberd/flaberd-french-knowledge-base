# L'imparfait (the imperfect)

## Usage

L'imparfait talks about the past, but **not** a precise, completed action (contrast with passé composé). It is used to:

1. **Describe a situation, state, or setting in the past** (context) — Il **faisait** froid ce jour-là. (It was cold that day.)
2. **Talk about a habit or repeated action in the past** ("used to") — Quand j'**étais** enfant, je **jouais** au foot tous les samedis. (When I was a child, I used to play football every Saturday.)
3. **Express an action that was in progress when another action happened** (past continuous) — Je **lisais** quand le téléphone a sonné. (I was reading when the phone rang.)

Common time markers/signal words: souvent, tous les jours, toujours, pendant que, d'habitude, quand j'étais petit(e), à l'époque.

## Formation

> Take the **nous** form of the present tense → remove **-ons** → add the imparfait endings.

1. Find the *nous* form in the present tense: parler → nous parl**ons**; finir → nous finiss**ons**; prendre → nous pren**ons**.
2. Remove **-ons** to get the stem: parlons → **parl**; finissons → **finiss**; prenons → **pren**.
3. Add the imparfait ending for the subject.

| Pronom | Terminaison |
|---|---|
| Je | -ais |
| Tu | -ais |
| Il/elle/on | -ait |
| Nous | -ions |
| Vous | -iez |
| Ils/elles | -aient |

### Spelling adjustments

- **-ger verbs**: an extra "e" is kept before -ais/-ait/-aient to preserve the soft "g" sound — je mange**ais**, je voyage**ais** (g + e + a).
- **-cer verbs**: ç replaces c before -ais/-ait/-aient to preserve the soft "c" sound — je commen**çais** (ç + a).

*(Same spelling-change logic as the present tense — see [grammar/er-verbs-stem-changes.md](er-verbs-stem-changes.md) — but the imparfait forms it affects are je/tu/il/ils, not nous/vous, since the nous/vous imparfait stem already has -i- absorbing the sound.)*

## Full conjugation by verb type

| Pronom | -er (aimer) | -ir, finissons-type (finir) | -ir, dormons-type (dormir) | -re (vendre) |
|---|---|---|---|---|
| je/j' | aimais | finissais | dormais | vendais |
| tu | aimais | finissais | dormais | vendais |
| il/elle/on | aimait | finissait | dormait | vendait |
| nous | aimions | finissions | dormions | vendions |
| vous | aimiez | finissiez | dormiez | vendiez |
| ils/elles | aimaient | finissaient | dormaient | vendaient |

The two "-ir" columns show the split already documented in [grammar/verb-groups.md](verb-groups.md): 2e groupe verbs like *finir* keep the -iss- infix (nous finiss**ons**), while irregular 3e groupe -ir verbs like *dormir* don't (nous dorm**ons**) — the imparfait stem simply follows whichever *nous* form the verb already has.

## Être: the only irregular verb in the imparfait

*Être* does **not** follow the "nous form minus -ons" rule (nous **sommes** has no -ons to remove — "je sommais" doesn't work). Its imparfait stem is irregular: **ét-**.

| Sujet | Forme imparfait |
|---|---|
| J' | étais |
| Tu | étais |
| Il/elle | était |
| Nous | étions |
| Vous | étiez |
| Ils/elles | étaient |

Every other verb, however irregular its present tense, forms the imparfait regularly from its own *nous* form (avoir → nous avons → av-; aller → nous allons → all-; faire → nous faisons → fais-, etc.).
