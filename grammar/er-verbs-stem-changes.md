# -ER verbs: stem-vowel changes

## Why these changes happen

French word stress always falls on the last syllable. If a weak [ə]/[e] sound were left in a stressed final stem syllable (i.e. everywhere except *nous*/*vous*), it would "collapse" and sound unclear. So -er verbs whose stem ends in a weak e-sound fix this one of two ways:

- **Double the following consonant** (appeler, jeter) → the vowel before it becomes a clear [ɛ].
- **Add a grave accent, e → è** (acheter, mener, préférer) → the weak [ə]/[e] becomes an open [ɛ].

Either way, the **nous** and **vous** forms keep the original unstressed vowel (no change), because the stress there falls on the ending, not the stem.

## Pattern 1: é/e → è (préférer-type)

| Pronom | PRÉFÉRER |
|---|---|
| Je | préfère |
| tu | préfères |
| il/elle/on | préfère |
| nous | préférons |
| vous | préférez |
| ils/elles | préfèrent |

Similar verbs: espérer, répéter, compléter, célébrer.

## Pattern 2: e → è (acheter-type)

| Pronom | ACHETER |
|---|---|
| J' | achète |
| tu | achètes |
| il/elle/on | achète |
| nous | achetons |
| vous | achetez |
| ils/elles | achètent |

Similar verbs: mener, amener, emmener, enlever, peser.

## Pattern 3: doubled consonant (jeter-type, appeler-type)

| Pronom | JETER | APPELER |
|---|---|---|
| Je | jette | appelle |
| tu | jettes | appelles |
| il/elle/on | jette | appelle |
| nous | jetons | appelons |
| vous | jetez | appelez |
| ils/elles | jettent | appellent |
