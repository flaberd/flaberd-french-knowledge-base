# Numbers 80 and above (formation irregularities)

Builds on [grammar/numbers-1-50.md](numbers-1-50.md) (which covers 1–50 and the general tens/hyphen/"et" pattern).

## Key irregularities highlighted in the source

| Number | French | Note |
|---|---|---|
| 80 | quatre-vingt**s** | Literally "four twenties"; takes an **-s** when it is a round multiple of twenty with nothing following it. |
| 81 | quatre-vingt-un | Unlike 21/31/41/51/61/71 ("et un"), 81 does **not** use "et": it's quatre-vingt-**un**, not ~~quatre-vingt-et-un~~. |
| 90 | quatre-vingt-dix | Literally "four twenties and ten"; continues from quatre-vingts + the teens pattern (dix, onze...). |
| 200 | deux cent**s** | "Cent" takes an **-s** when it is a round multiple of a hundred with nothing following it. |
| 202 | deux-cent-deux | The **-s** on "cents" is dropped as soon as another number follows it. |

## Formation pattern

- 70–79: soixante + dix/onze/douze... (soixante-dix, soixante et onze, soixante-douze...).
- 80–89: quatre-vingts (round) / quatre-vingt- + un..neuf (no "et" before "un").
- 90–99: quatre-vingt-dix (round) / quatre-vingt- + onze..dix-neuf.
- Multiples of 100 and 1000 (deux cents, trois cents...) take a plural **-s**; the **-s** disappears if another number follows (deux-cent-deux, trois-cent-cinq).
- "Mille" (thousand) is invariable — never takes an -s (see [vocabulary/numbers/mille.md](../vocabulary/numbers/mille.md)).

## Reference chart

The source includes a full reference chart (0–100, plus "les grands nombres" up to 2 000 000 000), attributed to Woodward French and written in the post-1990 rectified orthography (hyphens throughout, e.g. "deux-cent-deux"). Useful link given alongside it: https://www.languageguide.org/french/numbers/

## Notes

- Only the five values above (80, 81, 90, 200, 202) were explicitly called out ("!!") in the source; the rest of 51–99 and 101–999 are not individually documented as vocabulary, consistent with how the 1–50 range was introduced as a full dictated list while later ranges are practiced orally without a term-by-term teaching pass.
