# 2e groupe (-ir) verbs: present tense

## Conjugation pattern (grandir)

| Pronom | Terminaison | Exemple |
|---|---|---|
| Je | -is | je grandis |
| tu | -is | tu grandis |
| il / elle / on | -it | il grandit |
| nous | -issons | nous grandissons |
| vous | -issez | vous grandissez |
| ils/elles | -issent | ils grandissent |

This pattern is regular and applies to all 2e groupe verbs: finir, choisir, agir, grossir, maigrir, réussir, remplir, fournir, bâtir, investir, réfléchir, atterrir, applaudir, obéir, réunir, divertir, épanouir, réjouir, rafraîchir, accomplir, endurcir…

## Astuce — verbs formed from an adjective

Many 2e groupe verbs are built on an adjective and express a transformation ("becoming X"):

- vieille → vieillir → je vieillis → vous vieillissez
- rouge → rougir → je rougis → vous rougissez
- grande → grandir → je grandis → vous grandissez
- gros(se) → grossir → je grossis → vous grossissez

## Not every -ir verb is 2e groupe

A number of common -ir verbs (dormir, partir, sortir, sentir, servir, venir, courir, offrir…) are **irregular** and belong to the 3e groupe instead — they do not take the -issons/-issez/-issent endings. See [grammar/verb-groups.md](verb-groups.md). For example, *s'endormir* is built on *dormir* and is therefore 3e groupe, not 2e groupe, despite ending in -ir.
