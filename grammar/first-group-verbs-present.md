# 1er groupe verbs (-er) — present tense

## Conjugation pattern

| Pronom | Terminaison | Exemple PARLER |
|---|---|---|
| Je | -e | je parle |
| tu | -es | tu parles |
| il / elle / on | -e | il parle |
| nous | -ons | nous parlons |
| vous | -ez | vous parlez |
| ils/elles | -ent | ils parlent |

## Pronunciation

The endings -e, -es, and -ent are **silent**.

## Elision of "je"

The "e" of *je* becomes *j'* before a vowel or a mute *h*: je aime → **j'aime**, je habite → **j'habite**.

## -GER verbs: nous exception

Verbs ending in *-ger* insert an extra silent "e" before the *-ons* ending, to keep the "g" sound soft: **mangeons** (not *mangons*).

| Pronom | Terminaison | MANGER |
|---|---|---|
| Je | -e | mange |
| tu | -es | manges |
| il / elle / on | -e | mange |
| **nous** | **-ons** | **mangeons** |
| vous | -ez | mangez |
| ils/elles | -ent | mangent |

Other verbs with the same pattern: partager → nous partageons. nager → nous nageons. voyager → nous voyageons.

## -CER verbs: nous exception

Verbs ending in *-cer* take a cedilla before the *-ons* ending, to keep the "c" sound soft: **commençons** (not *commencons*).

| Pronom | Terminaison | COMMENCER |
|---|---|---|
| Je | -e | commence |
| tu | -es | commences |
| il / elle / on | -e | commence |
| **nous** | **-ons** | **commençons** |
| vous | -ez | commencez |
| ils/elles | -ent | commencent |

Other verbs with the same pattern: lancer → nous lançons. avancer → nous avançons.

## Negation

Subject + **ne** + verb + **pas**: je ne parle pas, tu ne parles pas, il ne parle pas, nous ne parlons pas, vous ne parlez pas, ils ne parlent pas. See [grammar/negation.md](negation.md).
