# Demonstrative adjectives: ce, cette, cet, ces

## Use

The demonstrative adjective points something out ("this/that"): *Regardez : ce chat / cette voiture / ces arbres.*

- Regarde **cette** fourmi (sur le sol).
- Regarde **cette** étoile (dans le ciel).

## Forms

The demonstrative agrees with the noun it accompanies:

| | Singular | Plural |
|---|---|---|
| Masculine | ce (cet before a vowel or mute h) | ces |
| Feminine | cette | ces |

- ce chat / cette voiture / ces arbres / ces voitures

## Elision: ce → cet

"Ce" becomes "cet" before a masculine noun starting with a vowel or a mute "h":

- ce acteur → **cet acteur** (feminine: cette actrice)
- ce homme → **cet homme**

## Pronunciation

- **ces** + vowel-initial noun → liaison in [z]: *ces_arbres*, *ces_hommes*.
