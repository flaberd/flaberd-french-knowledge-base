# Accents (les accents)

| Accent | Letters | Examples |
|---|---|---|
| Accent aigu ( ´ ) | é | été, école |
| Accent grave ( ` ) | à, è, ù | là, père, où |
| Cédille ( ¸ ) | ç | français, garçon |
| Accent circonflexe ( ˆ ) | â, ê, î, ô, û | forêt, hôtel, maître, sûr |
| Tréma ( ¨ ) | ë, ï, ü, ÿ | Noël, maïs, ambiguïté |
